# CrossChat
<img src="image.png">
简单部署的跨服聊天！
## 支持版本
1.12.2 - 1.21+（已测试）
## 配置设置

```yaml
# 本服务器名称，会作为前缀
server-name: "你的服务器"

# server 或 client
mode: "server"

# server 模式下监听的端口
listen-port: 20360

# client 模式下要连接的服务器
remote-host: "127.0.0.1"
remote-port: 20370
```
> 如您的服务器server设置端口 `20360`，请为client设置为 `20360`。
> 如果您有多台服务器，请您设置其中一个为server，其余为client，并校准`端口`
